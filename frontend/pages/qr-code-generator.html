📖 How to Use

  1. Option A - Scan QR Code: Use your phone to scan the QR code above and it will show the JSON payload
  2. Option B - Copy JSON: Click the "Copy JSON" button to copy the payload, then paste it into your Blancco workflow
  3. Paste into Blancco: In your Blancco workflow's Server Message configuration, replace the JSON with the copied payload
  4. Update deviceType: Make sure "deviceType": "macs" or "deviceType": "mobiles" matches your workflow
  5. Save and test: Run a test erasure to verify the payload is working
